---
title: "Reset Password"
openapi: "POST /api/auth/forget-password"
---

The Reset Password API endpoint handles user auth requests.

## Endpoint

`POST {{BASE_URL}}/api/auth/forget-password`

### Request

The request should include the following JSON body:

```json
{
  "email": "user.email@gmail.com",
  "code": 4050,
  "password": "UserPassword@123"
}
```

### Success Response

The Response should include the following JSON body:

```json
{
  "statusCode": 200,
  "message": "Password forget successfully",
  "data": {}
}
```

### Error Responses

If the provided OTP is not found in the database, the response might look like this:

```json
{
  "statusCode": 400,
  "message": "Incorrect code entered. Please enter again.",
  "data": {}
}
```

### Validation Error

If the request body does not contain a valid email, the response might include a validation error:

```json
{
  "statusCode": 409,
  "message": "Email must be a valid email address."
  "data": {}
}
```

### Server Error

If there's an internal server error, the response might look like this:

```json
{
  "statusCode": 500,
  "message": "Internal server error. Please try again later.",
  "data": {}
}
```
