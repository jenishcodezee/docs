The Company Bank Retrieval API is designed to handle requests for fetching the details of a specific company bank in the system. This API is part of the organization management module and is responsible for retrieving company bank details from the database.

## Endpoint

`GET {{BASE_URL}}/api/organization/bank/{bankId}`

### Bad Request Responses

Provided Request Server cannot or will not process the request due to something that is perceived to be a client error.

```json
{
  "statusCode": 400,
  "message": "Bad Request: Missing required fields",
  "data": {}
}
```

### Unauthenticated User Responses

If the request lacks proper authentication, the API responds might look like this:

```json
{
  "statusCode": 401,
  "message": "Token required",
  "data": {}
}
```

### Not found Responses

If the specified company bank is not found in the database, the response might look like this:

```json
{
  "statusCode": 404,
  "message": "Company bank not found.",
  "data": {}
}
```

### Server Error

If there is an internal server error, the response might look like this:

```json
{
  "statusCode": 500,
  "message": "Internal server error. Please try again later.",
  "data": {}
}
```
