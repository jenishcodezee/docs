title: "Delete"
openapi: "DELETE /api/organization/bank/{bankId}"
---

The Company Bank Deletion API is designed to handle requests for deleting a specific company bank account from the system. This API is part of the organization management module and is responsible for removing company bank details from the database.

## Endpoint

`DELETE {{BASE_URL}}/api/organization/bank/{locationId}`

### Success Response

On successful deletion of the company bank account, the API responds with the following JSON body:

```json
{
  "statusCode": 200,
  "message": "Company bank account deleted successfully.",
  "data": {}
}
```

### Unauthenticated User Responses

If the request lacks proper authentication, the response might look like this:

```json
{
  "statusCode": 401,
  "message": "Token required",
  "data": {}
}
```

### Data Not Found Responses

If the specified company bank account is not found in the database, the response might look like this:

```json
{
  "statusCode": 404,
  "message": "Company bank not found.",
  "data": {}
}
```

### Server Error

If there is an internal server error, the API responds might look like this:

```json
{
  "statusCode": 500,
  "message": "Internal server error. Please try again later.",
  "data": {}
}
```
