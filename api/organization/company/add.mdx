### Error Responses

If the provided email or password is not found in the database, the response might look like this:

```json
{
  "statusCode": 400,
  "message": "Incorrect credentials.",
  "data": {}
}
```

### Validation Error

If the request body does not contain a valid email, the response might include a validation error:

```json
{
  "statusCode": 409,
  "message": "Validation Error",
  "data": "Please provide a company name."
}
```

### Server Error

If there's an internal server error, the response might look like this:

```json
{
  "statusCode": 500,
  "message": "Internal server error. Please try again later.",
  "data": {}
}
```
