title: "List"
openapi: "POST /api/organization/company/list"
---

This API endpoint fetches a list of all companies associated with a particular organization. It returns a structured list containing details about each company.

## Endpoint

`POST {{BASE_URL}}/api/organization/company/list`

### Request

To register a new company, the client must send a POST request to the specified endpoint with the following JSON payload:

```json
{
  "page": 1,
  "limit": 15,
  "search": ""
}
```

### Unauthenticated User Responses

If the provided company is not found in the database, the response might look like this:

```json
{
  "statusCode": 401,
  "message": "Token required",
  "data": {}
}
```

### Validation Error

If the request body does not contain a valid email, the response might include a validation error:

```json
{
  "statusCode": 409,
  "message": "Validation Error",
  "data": "Limit must be greater than or equal to 1"
}
```

### Server Error

If there's an internal server error, the response might look like this:

```json
{
  "statusCode": 500,
  "message": "Internal server error. Please try again later.",
  "data": {}
}
```
