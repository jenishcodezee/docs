---
title: "Delete"
openapi: "DELETE /api/organization/company/{id}"
---

This API endpoint is used to delete a specific company from the organization. The company to be deleted is identified by its unique companyId.

## Endpoint

`DELETE {{BASE_URL}}/api/organization/company/{id}`

### Success Response

The Response should include the following JSON body:

```json
{
  "statusCode": 200,
  "message": "Company deleted successfully.",
  "data": {}
}
```

### Unauthenticated User Responses

If the request lacks proper authentication, the API responds might look like this:

```json
{
  "statusCode": 401,
  "message": "Token required",
  "data": {}
}
```

### Data Not Found Responses

If the provided email or password is not found in the database, the response might look like this:

```json
{
  "statusCode": 404,
  "message": "Company not found.",
  "data": null
}
```

### Server Error

If there is an internal server error, the API responds might look like this:

```json
{
  "statusCode": 500,
  "message": "Internal server error. Please try again later.",
  "data": {}
}
```
