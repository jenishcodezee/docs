---
title: "Delete"
openapi: "DELETE /api/organization/business-location/{locationId}"
---

The Business Location Deletion API is designed to handle requests for deleting a specific business location from the system. This API is part of the organization management module and is responsible for removing business location details from the database.

## Endpoint

`DELETE {{BASE_URL}}/api/organization/business-location/{locationId}`

### Success Response

On successful deletion of the business location, the API responds with the following JSON body:

```json
{
  "statusCode": 200,
  "message": "Business location deleted successfully.",
  "data": {}
}
```

### Unauthenticated User Responses

If the request lacks proper authentication, the response might look like this:

```json
{
  "statusCode": 401,
  "message": "Token required",
  "data": {}
}
```

### Data Not Found Responses

If the specified business location is not found in the database, the response might look like this:

```json
{
  "statusCode": 404,
  "message": "Business location not found.",
  "data": null
}
```

### Server Error

If there is an internal server error, the API responds might look like this:

```json
{
  "statusCode": 500,
  "message": "Internal server error. Please try again later.",
  "data": {}
}
```
