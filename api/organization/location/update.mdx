---
title: "Update"
openapi: "PUT /api/organization/business-location/{locationId}"
---

The Business Location Update API is designed to handle requests for updating the details of an existing business location in the system. This API is part of the organization management module and is responsible for modifying business location details in the database.

## Endpoint

`PUT {{BASE_URL}}/api/organization/business-location/{locationId}`

### Request

The request should include the following JSON body:

```json
{
  "business_unit_id": "25b5d8d0-eea7-44e8-a744-55f124bf0d3e",
  "location_name": "Samsung Enterprises",
  "email": "info@samsungenterprises.com",
  "phone_number": "9632587410",
  "mobile_number": "9632587410",
  "address": "1234 Samsung St, Suite 100",
  "state": "9f89210f-14bf-4451-93e8-d41a0963df8e",
  "country": "cb6c8d39-ab05-4220-90d6-8eda6b3b9ce5",
  "pincode": "364587"
}
```

### Bad Request Responses

Provided Request Server cannot or will not process the request due to something that is perceived to be a client error.

```json
{
  "statusCode": 400,
  "message": "Bad Request: Missing required fields",
  "data": {}
}
```

### Unauthenticated User Responses

If the request lacks proper authentication, the API responds might look like this:

```json
{
  "statusCode": 401,
  "message": "Token required",
  "data": {}
}
```

### Data Not Found Responses

If the provided business location is not found in the database, the response might look like this:

```json
{
  "statusCode": 404,
  "message": "Business location not found.",
  "data": {}
}
```

### Validation Error

If the request body does not contain valid data, the response might include a validation error:

```json
{
  "statusCode": 409,
  "message": "Validation Error",
  "data": "Please provide a location name."
}
```

### Server Error

If there is an internal server error, the API responds might look like this:

```json
{
  "statusCode": 500,
  "message": "Internal server error. Please try again later.",
  "data": {}
}
```
