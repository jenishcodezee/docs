The Business Location Listing API is designed to handle requests for fetching a list of business locations within the organization from the system. This API is part of the organization management module and is responsible for retrieving multiple business location details from the database.

## Endpoint

`POST {{BASE_URL}}/api/organization/business-location/list`

### Request

To fetch a list of business locations, the client must send a POST request to the specified endpoint.

```json
{
  "page": 1,
  "limit": 15,
  "search": ""
}
```

### Unauthenticated User Responses

If the request lacks proper authentication, the API responds might look like this:

```json
{
  "statusCode": 401,
  "message": "Token required",
  "data": {}
}
```

### Validation Error

If the request body contains invalid data, the response might include a validation error:

```json
{
  "statusCode": 409,
  "message": "Validation Error",
  "data": "Limit must be greater than or equal to 1"
}
```

### Server Error

If there is an internal server error, the API responds might look like this:

```json
{
  "statusCode": 500,
  "message": "Internal server error. Please try again later.",
  "data": {}
}
```
