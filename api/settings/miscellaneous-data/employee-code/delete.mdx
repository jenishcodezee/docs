---
title: "Delete"
openapi: "DELETE /api/settings/employee-code/{employeeCodeId}"
---

The Delete Employee Code API allows clients to delete an existing employee code from the system. This API is part of the settings management module and is responsible for removing employee code details from the database.

## Endpoint

`DELETE {{BASE_URL}}/api/settings/employee-code/{employeeCodeId}`

### Success Response

On successful deletion of the employee code, the API responds with the following JSON body:

```json
{
  "statusCode": 200,
  "message": "Employee code deleted successfully.",
  "data": {}
}
```

### Bad Request Responses

If the server cannot process the request due to a client error, the response might look like this:

```json
{
  "statusCode": 400,
  "message": "Bad Request: Invalid required fields",
  "data": {}
}
```

### Unauthenticated User Responses

If the request lacks proper authentication, the API responds with the following JSON body:

```json
{
  "statusCode": 401,
  "message": "You are an unauthorized user",
  "data": {}
}
```

### Data Not Found Responses

If the specified employee code ID is not found in the database, the response might look like this:

```json
{
  "statusCode": 404,
  "message": "Employee code not found.",
  "data": {}
}
```

### Server Error

If there is an internal server error, the API responds with the following JSON body:

```json
{
  "statusCode": 500,
  "message": "Internal server error. Please try again later.",
  "data": {}
}
```
