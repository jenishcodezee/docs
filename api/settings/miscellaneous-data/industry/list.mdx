The Industries List API allows clients to retrieve a filtered list of industries from the system. This API is part of the settings management module and supports querying and pagination for industries.

## Endpoint

`POST {{BASE_URL}}/api/settings/industries/list`

### Request

To retrieve a list of industries, send a POST request to the endpoint with optional query parameters:

```json
{
  "page": 1,
  "limit": 15,
  "is_active": true,
  "search": "ere"
}
```

### Unauthenticated User Responses

If the request lacks proper authentication, the API response might look like this:

```json
{
  "statusCode": 401,
  "message": "Token required",
  "data": {}
}
```

### Validation Error

If the request body contains invalid data, the response might include a validation error:

```json
{
  "statusCode": 409,
  "message": "Validation Error",
  "data": "is_active must be true or false."
}
```

### Server Error

If there is an internal server error, the API responds might look like this:

```json
{
  "statusCode": 500,
  "message": "Internal server error. Please try again later.",
  "data": {}
}
```
